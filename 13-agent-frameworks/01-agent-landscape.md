# Agent Landscape

## Purpose

Define the roles required for enterprise knowledge discovery.

## Agents

| Agent | Responsibility |
|---------|----------------|
| Discovery Agent | Extract observations from artefacts |
| Evidence Agent | Convert observations into evidence |
| Dictionary Agent | Build canonical terminology |
| Architecture Agent | Derive architecture models |
| Consistency Agent | Validate repository quality |

---

## Design Principle

Each agent has a single responsibility.

Agents do not replace human judgement.

Agents support knowledge creation.
