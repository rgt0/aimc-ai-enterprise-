# Discovery Agent

## Purpose

Transform raw enterprise artefacts into observations.

## Inputs

- XINFO
- Scheduler
- JCL
- COBOL source
- Section inventories
- DB2 metadata

## Outputs

Observations

## Rules

- Record what is visible
- Do not create architecture
- Do not infer business meaning
- Avoid assumptions

## Example

Input

BL755A00 Header

Output

Observation:

Program generates tolerance reports.

Classification:

OBSERVATION
