# ASKT as an AI System

ASKT should not be viewed merely as a chat interface.

From an AIMC perspective, ASKT is an enterprise AI system that provides access to a collection of specialized AI capabilities.

Historically these capabilities appeared as:

- Prompts
- Apps
- Assistants

ASKT has unified these concepts under a common abstraction:

Agent

This evolution reflects a broader enterprise AI trend where AI capabilities are packaged as reusable systems rather than individual prompts.

---

## Main Concepts

### Agent

An Agent combines:

- role
- instructions
- knowledge
- tools
- execution logic

into a reusable capability.

### Agent Library

ASKT provides a library of Agents organized around common business tasks.

Users select an Agent rather than constructing prompts from scratch.

---

Traditional model:

User

User

---

## Why It Matters

The Agent model:

- simplifies discovery
- improves usability
- promotes reuse
- enables governance
- supports enterprise adoption

From an AIMC perspective, Agents are building blocks of enterprise AI systems.
